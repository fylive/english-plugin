# 🚀 快速开始

## 一分钟安装指南

### 1️⃣ 安装插件

**Chrome:**
```
1. 打开 chrome://extensions/
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本文件夹
```

**Edge:**
```
1. 打开 edge://extensions/
2. 开启"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择本文件夹
```

### 2️⃣ 测试功能

1. 在浏览器中打开 `test.html` 文件
2. 选中任意中文文本
3. 点击出现的紫色翻译图标
4. 查看翻译结果

### 3️⃣ 添加图标（可选）

1. 在浏览器中打开 `icon-generator.html`
2. 按照说明截图生成图标
3. 将图标文件放入 `icons/` 文件夹
4. 在扩展管理页面重新加载插件

**注意：** 没有图标也可以正常使用，只是显示默认图标。

## 🔧 常见问题

**Q: 图标不显示怎么办？**
A: 图标是可选的，不影响功能使用。如需添加，参考 `icon-generator.html`。

**Q: 翻译失败怎么办？**
A: 检查网络连接，或在popup中配置自己的翻译API。

**Q: 某些网站无法使用？**
A: 部分网站有CSP限制，这是正常现象。
