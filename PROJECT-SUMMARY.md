## 📦 安装步骤

### Chrome浏览器
```
1. 打开 chrome://extensions/
2. 启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目文件夹
```

### Edge浏览器
```
1. 打开 edge://extensions/
2. 启用"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择项目文件夹
```
