# 中英翻译助手 Chrome/Edge 插件

一个简洁高效的浏览器扩展，帮助你快速翻译网页中的中文内容为英文。

## 功能特点

- ✅ **智能识别**：自动识别网页中的中文文本
- ✅ **即时翻译**：选中文本后一键翻译成英文
- ✅ **优雅界面**：美观的悬浮翻译面板
- ✅ **全网支持**：适用于所有网页
- ✅ **免费使用**：使用免费翻译API，无需注册
- ✅ **自定义API**：支持配置自己的翻译API

## 安装方法

### Chrome 浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

### Edge 浏览器

1. 打开 Edge 浏览器
2. 在地址栏输入 `edge://extensions/`
3. 打开左下角的"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择本项目文件夹

## 使用方法

1. 在任意网页上选中中文文本
2. 会自动出现一个紫色的翻译图标
3. 点击图标查看翻译结果
4. 点击面板右上角的 × 关闭翻译面板

## 高级配置

如果你有自己的翻译API（如DeepL、OpenAI等），可以在插件弹窗中配置：

1. 点击浏览器工具栏中的插件图标
2. 在"高级设置"中输入你的API地址和密钥
3. 点击"保存设置"

## 技术实现

- **Manifest V3**：使用最新的Chrome扩展规范
- **Content Script**：在网页中注入脚本识别和处理文本
- **免费API**：默认使用Google Translate的免费接口
- **备用方案**：支持MyMemory等多个翻译API作为备选
