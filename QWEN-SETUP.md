# 🤖 通义千问配置指南

- ✅ **无需翻墙** - 国内直接访问

## 📝 配置步骤

### 1. 获取API密钥

1. **访问通义千问控制台**
   - 打开：https://dashscope.console.aliyun.com/

2. **登录阿里云账号**
   - 如果没有账号，先注册
   - 可以使用支付宝快速登录

3. **创建API密钥**
   - 进入"API-KEY管理"页面
   - 点击"创建新的API-KEY"
   - 复制生成的密钥（格式：sk-...）

4. **开通服务**
   - 确保已开通"通义千问"服务
   - 新用户通常有免费额度

### 2. 配置插件

1. **打开插件设置**
   - 点击浏览器工具栏的插件图标

2. **选择AI智能模式**
   - 在"选词模式"中选择"AI智能"

3. **选择通义千问**
   - 在"AI服务"下拉框中选择"通义千问（推荐）"

4. **输入API密钥**
   - 粘贴你的通义千问API密钥
   - 格式：sk-...

5. **应用设置**
   - 点击"应用设置并刷新页面"

### 3. 开始使用

1. **刷新页面**
   - 页面会自动刷新

2. **查看效果**
   - AI会分析页面内容
   - 智能选择合适的词汇替换
   - 首次加载需要1-3秒

3. **查看日志**
   - 按F12打开控制台
   - 查看AI分析过程
