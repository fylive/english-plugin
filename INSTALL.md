# 安装和使用指南

## 📦 安装步骤

### Chrome 浏览器安装

1. **打开扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者：点击浏览器右上角的三个点 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开关，启用开发者模式

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择本项目的文件夹（包含manifest.json的文件夹）
   - 点击"选择文件夹"

4. **完成安装**
   - 扩展会出现在扩展列表中
   - 可以点击"固定"图标将扩展固定到工具栏

### Edge 浏览器安装

1. **打开扩展管理页面**
   - 在地址栏输入：`edge://extensions/`
   - 或者：点击浏览器右上角的三个点 → 扩展

2. **启用开发人员模式**
   - 在页面左下角找到"开发人员模式"开关
   - 点击开关，启用开发人员模式

3. **加载扩展**
   - 点击"加载解压缩的扩展"按钮
   - 选择本项目的文件夹（包含manifest.json的文件夹）
   - 点击"选择文件夹"

4. **完成安装**
   - 扩展会出现在扩展列表中
   - 可以点击"固定"图标将扩展固定到工具栏

## 🎯 使用方法

### 基本使用

1. **打开任意网页**
   - 访问包含中文内容的网页（如新闻网站、博客等）

2. **选择文本**
   - 用鼠标选中你想翻译的中文文本
   - 松开鼠标后，会在选中文本上方出现一个紫色的翻译图标

3. **查看翻译**
   - 点击紫色的翻译图标
   - 会弹出翻译面板，显示原文和译文

4. **关闭面板**
   - 点击翻译面板右上角的 × 按钮关闭

### 高级设置（可选）

如果你有自己的翻译API（如DeepL、OpenAI等），可以配置自定义API：

1. **打开设置**
   - 点击浏览器工具栏中的插件图标
   - 会打开设置弹窗

2. **配置API**
   - 在"自定义API地址"中输入你的API端点
   - 在"API密钥"中输入你的密钥
   - 点击"保存设置"

3. **使用自定义API**
   - 保存后，插件会优先使用你配置的API进行翻译

## 🔧 故障排除

### 问题1：图标不显示

**原因：** 缺少图标文件

**解决方案：**
- 打开 `icon-generator.html` 文件
- 按照说明生成图标文件
- 或者暂时忽略，不影响功能使用

### 问题2：翻译失败

**可能原因：**
- 网络连接问题
- 免费API达到请求限制
- 网站的CSP策略限制

**解决方案：**
- 检查网络连接
- 等待一段时间后重试
- 配置自己的翻译API

### 问题3：某些网站无法使用

**原因：** 某些网站有严格的内容安全策略（CSP）

**解决方案：**
- 这是正常现象，部分网站会限制扩展功能
- 可以在其他网站上正常使用

### 问题4：翻译按钮位置不对

**原因：** 网页布局复杂或有固定定位元素

**解决方案：**
- 尝试滚动页面后重新选择文本
- 这不影响翻译功能的使用

## 📝 使用技巧

1. **快速翻译**
   - 选中文本后立即点击图标，无需等待

2. **长文本翻译**
   - 支持翻译较长的段落
   - 建议每次翻译不超过500字以获得最佳效果

3. **保持面板打开**
   - 翻译面板不会自动关闭
   - 可以对比查看原文和译文

4. **多次翻译**
   - 关闭面板后可以继续选择其他文本翻译
   - 每次翻译都是独立的

## 🎨 自定义图标（可选）

如果你想要更个性化的图标：

1. 使用设计工具（Figma、Photoshop等）创建图标
2. 导出为PNG格式，尺寸为16x16、48x48、128x128
3. 保存到 `icons/` 文件夹
4. 在扩展管理页面点击"重新加载"按钮

## 🔄 更新扩展

当你修改了代码或配置后：

1. 打开扩展管理页面
2. 找到"中英翻译助手"
3. 点击"重新加载"按钮（刷新图标）
4. 刷新正在使用的网页

## ⚠️ 注意事项

1. **API限制**
   - 默认使用的免费API可能有请求频率限制
   - 建议配置自己的API以获得更好的体验

2. **隐私保护**
   - 翻译的文本会发送到翻译API服务器
   - 请勿翻译敏感或机密信息

3. **网络要求**
   - 需要网络连接才能使用翻译功能
   - 确保可以访问翻译API服务

4. **浏览器兼容性**
   - 支持Chrome 88+和Edge 88+
   - 使用Manifest V3规范
