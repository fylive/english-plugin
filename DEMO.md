# 🎬 使用演示

## 第一步：安装插件

### Windows用户快捷方式
双击 `start.bat` 文件，选择对应的选项快速打开相关页面。

### 手动安装

**Chrome浏览器：**
1. 地址栏输入：`chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹

**Edge浏览器：**
1. 地址栏输入：`edge://extensions/`
2. 开启左下角的"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择本项目文件夹

## 第二步：测试功能

### 使用测试页面
1. 双击打开 `test.html` 文件
2. 或运行 `start.bat` 选择选项 [1]

### 测试步骤
1. 在测试页面中，用鼠标选中任意中文文本
2. 松开鼠标后，会在文本上方出现一个紫色的圆形翻译图标
3. 点击这个图标
4. 会弹出一个翻译面板，显示原文和译文
5. 点击面板右上角的 × 关闭面板

### 预期效果

**选中文本时：**
```
你好，世界！
    ↑
   [🌐] ← 紫色翻译图标出现
```

**点击图标后：**
```
┌─────────────────────────┐
│ 翻译              ×     │
├─────────────────────────┤
│ 原文：                  │
│ 你好，世界！            │
│                         │
│ 译文：                  │
│ Hello, world!           │
└─────────────────────────┘
```

## 第三步：实际使用

### 在任意网页使用
1. 访问任何包含中文的网页（如新闻网站、博客等）
2. 选中你想翻译的中文文本
3. 点击出现的翻译图标
4. 查看翻译结果

### 推荐测试网站
- 百度新闻：https://news.baidu.com/
- 知乎：https://www.zhihu.com/
- 简书：https://www.jianshu.com/
- 微信公众号文章
- 任何包含中文的网页

## 第四步：配置设置（可选）

### 打开设置
1. 点击浏览器工具栏中的插件图标
2. 会弹出设置界面

### 配置自定义API
如果你有自己的翻译API：
1. 在"自定义API地址"输入框中输入API端点
2. 在"API密钥"输入框中输入你的密钥
3. 点击"保存设置"按钮

### 默认配置
如果不配置自定义API，插件会使用：
- 主要：Google Translate 免费接口
- 备用：MyMemory 翻译API

## 界面元素说明

### 翻译按钮
- **外观**：紫色圆形按钮，带有翻译图标
- **位置**：选中文本上方
- **交互**：鼠标悬停时会放大
- **功能**：点击显示翻译面板

### 翻译面板
- **标题栏**：紫色渐变背景，显示"翻译"和关闭按钮
- **原文区域**：灰色背景，显示选中的原文
- **译文区域**：绿色背景，显示翻译结果
- **关闭按钮**：右上角的 × 按钮

## 常见使用问题

### Q1: 选中文本后没有出现翻译按钮？
**检查项：**
- ✅ 确认插件已安装并启用
- ✅ 确认选中的文本包含中文字符
- ✅ 刷新网页后重试
- ✅ 检查浏览器控制台是否有错误

### Q2: 点击按钮后翻译失败？
**可能原因：**
- 网络连接问题
- API请求限制
- 网站CSP限制

**解决方法：**
- 检查网络连接
- 等待几秒后重试
- 尝试在其他网站使用

### Q3: 翻译结果不准确？
**说明：**
- 使用的是免费翻译API，准确度有限
- 建议配置专业的翻译API（如DeepL、OpenAI）

### Q4: 某些网站无法使用？
**说明：**
- 部分网站有严格的内容安全策略（CSP）
- 这是正常现象，不影响在其他网站使用

## 使用技巧

### 技巧1：快速翻译
选中文本后立即点击图标，无需等待。

### 技巧2：对比阅读
翻译面板不会自动关闭，可以对比查看原文和译文。

### 技巧5：专业术语
对于专业术语，建议配置专业的翻译API。
